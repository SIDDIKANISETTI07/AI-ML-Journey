  1. day3_pattern.py

   Prints a left-aligned triangle pattern of * using nested for loops

   * *
   * * *
   * * * *

  2. day3_fizzbuzz.py
   Prints numbers from 1 to 100
   For multiples of 3 → prints "Fizz"
   For multiples of 5 → prints "Buzz"
   For multiples of both 3 and 5 → prints "FizzBuzz"
